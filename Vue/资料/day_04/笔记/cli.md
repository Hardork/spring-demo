###vue-cli的使用

    1.在指定目录下,使用cli创建项目
        vue create 项目名称
    
    2.在配置完成后,cd进入项目文件夹
        npm run serve
    
    3.vue中src目录的构成:
        
        1.assets 文件夹: 存放项目中用到的静态资源,例如: css样式表、图片文件

        2.components 文件夹: 程序员封装的、可复用的组件，都要放到 components 目录下

        3.main.js 是项目的入口文件,整个项目的进行,要先运行main.js

        4.app.vue是项目的根组件。
